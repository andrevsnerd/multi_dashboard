========================================
VARIÁVEIS PARA CONFIGURAR NO VERCEL
========================================

IMPORTANTE: Configure estas variáveis ANTES de fazer o deploy!

1. PROXY_URL
   Valor: (URL do ngrok - você verá quando iniciar o ngrok)
   Exemplo: https://abc123.ngrok-free.app
   
   ⚠️  Esta URL muda a cada vez que você reinicia o ngrok (plano gratuito)

2. PROXY_SECRET
   Valor: proxy-nerd-2024-1591
   
   ✅ Este valor está no seu .env.local

3. NODE_ENV
   Valor: production

========================================
COMO CONFIGURAR NO VERCEL:
========================================

1. Acesse seu projeto no Vercel
2. Vá em Settings → Environment Variables
3. Para cada variável acima:
   - Clique em "Add"
   - Coloque o Key e Value
   - Marque: ✅ Production, ✅ Preview, ✅ Development
   - Clique em "Save"

IMPORTANTE: 
- NÃO adicione as variáveis do banco (DB_SERVER, etc.) quando usar proxy
- Apenas PROXY_URL, PROXY_SECRET e NODE_ENV são necessárias

========================================
APÓS CONFIGURAR:
========================================

1. Faça um novo deploy no Vercel
2. Mantenha o proxy e ngrok rodando localmente
3. Teste o app!
